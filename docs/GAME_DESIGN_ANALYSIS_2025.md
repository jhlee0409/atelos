### 2.2 플레이어 행동 분석

**플레이어가 실제로 하는 것:**

| 행동 | 빈도 | 소요 시간 | 인지 부하 |
|------|------|-----------|-----------|
| 텍스트 읽기 | 매 턴 | 15-30초 | 중간 |
| 선택지 비교 | 매 턴 | 5-10초 | 낮음 |
| 버튼 클릭 | 매 턴 | 1초 | 최소 |
| 스탯 확인 | 가끔 | 3-5초 | 낮음 |
| 대기 | 매 턴 | 1-3초 | 없음 |

**플레이어가 하지 않는 것:**
- ❌ 탐색 (맵, 환경, 오브젝트)
- ❌ 자원 관리 (인벤토리, 시간)
- ❌ 캐릭터 상호작용 (대화 선택, 관계 빌딩)
- ❌ 전략 수립 (목표 설정, 계획)
- ❌ 스킬/능력 사용
- ❌ 자유 텍스트 입력

### 2.3 세션 통계 (추정)

```
평균 플레이 시간: 30-60분
평균 턴 수: 15-25턴
턴당 평균 시간: 2분
선택지 총 수: 30-50개 (A/B × 턴 수)
실제 분기점: 이론적 2^n, 실질적으로 3개 루트로 수렴
엔딩 종류: 5-8개 (시나리오별)
```

```
2023년: AI 텍스트 생성 게임 등장 (AI Dungeon 류)

  - 자유도 높은 플레이어 입력 처리
  - 실시간 AI 협력 스토리텔링
  - 투명한 AI 의사결정 시각화
  - 감정 인식 기반 적응형 서사
```

### 3.2 경쟁 요소 비교

| 요소 | 2025 트렌드 | ATELOS 현재 | 격차 |
|------|-------------|-------------|------|
| **선택지 수** | 3개 이상 + 자유 입력 | 2개 고정 | 🔴 큼 |
| **AI 역할** | 협력적 공동 창작자 | 일방적 서술자 | 🟡 중간 |
| **플레이어 입력** | 자유 텍스트 / 음성 | 버튼 클릭 | 🔴 큼 |

| **캐릭터 깊이** | 대화 가능한 AI NPC | 수동적 조연 | 🟡 중간 |

## 4. 핵심 문제점

### 4.1 치명적 문제: 이진 선택의 한계

#### 문제 설명

모든 딜레마가 **A vs B** 구조로 고정되어 있음:

```typescript
// 현재 구조
dilemma: {
  prompt: "상황 설명",
  choice_a: "선택지 A",
  choice_b: "선택지 B"
  // choice_c 없음
  // 자유 입력 없음
}
```

#### 심리적 영향

| 상황 | 플레이어 감정 | 결과 |
|------|---------------|------|
| 둘 다 좋아 보임 | 결정 고민 | 긍정적 (정상) |
| 둘 다 싫어 보임 | **좌절감, 갇힌 느낌** | 부정적 |
| 하나만 명확히 좋음 | 선택이 뻔함 | 지루함 |
| 의미 차이 불분명 | 무작위 클릭 | 무관심 |

#### 경쟁작 비교

```
Disco Elysium: 4-8개 선택지 + 스킬 체크 옵션
┌─────────────────────────────────────────┐
│ [1] 직접적으로 물어본다 (권위)          │
│ [2] 에둘러 접근한다 (공감)              │
│ [3] 위협한다 (물리)                     │
│ [4] 뇌물을 제안한다                     │
│ [5] 그냥 지나친다                       │
│ [6] [논리 체크] 거짓말을 간파한다       │
└─────────────────────────────────────────┘

AI Dungeon: 자유 텍스트 입력
┌─────────────────────────────────────────┐
│ > 무엇을 하시겠습니까?                  │
│ [________________________] [전송]       │
│                                         │
│ 또는:                                   │
│ [A] 제안된 행동 1                       │
│ [B] 제안된 행동 2                       │
└─────────────────────────────────────────┘

ATELOS (현재):
┌─────────────────────────────────────────┐
│ [A] 선택지 A                            │
│ [B] 선택지 B                            │
│                                         │
│ (다른 옵션 없음)                        │
└─────────────────────────────────────────┘
```

### 4.2 높은 문제: 피드백 불투명성

#### 숨겨진 시스템들

```
플레이어가 모르는 것:
├── 스탯 증폭 시스템 (1.2x ~ 3.0x)
│   └── 중간값: 변화 크게 증폭
│   └── 극단값: 변화 완화
├── 관계 변화 상세
│   └── 어떤 캐릭터와의 관계가 변했는지
│   └── 왜 변했는지
├── 엔딩 조건
│   └── Day 5 전까지 완전 숨김
│   └── 어떤 스탯/플래그가 필요한지
├── 루트 시스템
│   └── Day 3 전까지 "미정" 표시
│   └── 어떤 선택이 어떤 루트로 가는지
└── 캐릭터 특성 영향
    └── 특성이 서사에 어떻게 반영되는지
```

#### 현재 vs 이상적 피드백

```
현재:
┌─────────────────────────────────────────┐
│ [선택 후]                               │
│                                         │
│ "당신은 협상을 시도했다..."             │
│ (서사 계속...)                          │
│                                         │
│ [스탯 바 조용히 업데이트]               │
└─────────────────────────────────────────┘

이상적:
┌─────────────────────────────────────────┐
│ [선택 후]                               │
│                                         │
│ "당신은 협상을 시도했다..."             │
│ (서사 계속...)                          │
│                                         │
│ ┌─ 결과 ─────────────────────────────┐ │
│ │ 📉 도시 혼란도 -5 (-10 증폭됨)     │ │
│ │ 📈 공동체 결속력 +3                │ │
│ │ 💔 박준경 신뢰 -2 (갈등 발생)      │ │
│ │ 🏳️ 협상 루트 진행도: 45% → 52%    │ │
│ └────────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

## 6. 개선 제안

### 6.1 선택지 시스템 확장

#### 제안 A: 3개 선택지 + 대기 옵션

```typescript
// 현재
dilemma: {
  prompt: string;
  choice_a: string;
  choice_b: string;
}

// 제안
dilemma: {
  prompt: string;
  choices: [
    { id: 'active', text: '적극적 행동', risk: 'high' },
    { id: 'cautious', text: '신중한 접근', risk: 'medium' },
    { id: 'wait', text: '상황을 지켜본다', risk: 'low' }
  ]
}
```

**구현 복잡도**: 중간
**예상 효과**: 플레이어 agency 30% 향상

```typescript

│                                         │
│ 또는 직접 입력:                         │
│ [______________________________] [전송] │
```

**구현 복잡도**: 높음 (AI 프롬프트 수정 필요)
